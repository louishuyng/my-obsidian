- You buy or register your domain name with a Domain Registrar typically by paing annual charges (eg GoDaddy, Amazon Registrar Inc., ...)
- The Domain Registrar usually provides you with a DNS service to manage your DNS records
- But you can use another DNS service to manage your DNS records
- If you buy your domain on a 3rd party registrar, you can still use Route 53 as the DNS Service provider
	1. Create a Hosted Zone in Route 53
	2. Update NS Records on 3rd website to use Route 53 Name Servers
- Domain Registrar != DNS Service
- Buy every Domain Registrar usually comes with some DNS features
